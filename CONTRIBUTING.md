# Contributing

Thanks for contributing to the Komga SDK.

## Prerequisites

- Bun (recommended)
- Node 18+

## Setup

```bash
bun install
```

## Development

```bash
bun run typecheck
bun run test
bun run test:coverage
```

## Documentation

```bash
bun run docs
```

## Guidelines

- Keep changes focused and add tests where appropriate.
- Update README and docs for any public API changes.
- Avoid using deprecated endpoints.

## Pull Requests

- Include a clear description of changes.
- Ensure tests and typecheck pass.
