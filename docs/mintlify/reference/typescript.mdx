---

This SDK is source-only and ships ESM. Use these `tsconfig.json` settings for best results.

```json
{
  "compilerOptions": {
    "module": "ESNext",
    "moduleResolution": "bundler",
    "target": "ES2020",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true
  }
}
```

## Notes

- Ensure your runtime supports Fetch and Web Crypto (Node 18+ or modern browsers).
- Prefer ESM imports in your app.
