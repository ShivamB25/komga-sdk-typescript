description: "Komga domain terms used throughout the SDK"
---

## Library

A collection of files indexed by Komga. Libraries are scanned and contain series and books.

## Series

A grouped set of books, usually ordered. Series contain one or more books.

## Book

A single item (CBZ, CBR, PDF, EPUB) within a series or library.

## Collection

A user-curated set of series.

## Read list

A user-curated set of books in reading order.

## Metadata

Descriptive information about a book or series, such as title, summary, and publisher.

## Scan

The process Komga uses to index files in a library and update metadata.

## Page

An individual image/page inside a book.
